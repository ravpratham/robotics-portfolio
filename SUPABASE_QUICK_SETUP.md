## What You Need To Do:

### 1. Create Supabase Project
- Go to: https://supabase.com
- Sign in and click "New Project"
- Name: robotics-portfolio
- Wait for creation (1-2 minutes)

### 2. Create Database Table
In Supabase Dashboard:
1. Go to **SQL Editor**
2. Click **New query**
3. Run this SQL:

```sql
create table assignments (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  title text not null,
  description text,
  section text not null,
  content text,
  youtube_url text
);

alter table assignments enable row level security;

create policy "Enable read access for all users" on assignments
  for select using (true);
  
create policy "Enable insert for authenticated users only" on assignments
  for insert to authenticated with check (true);

create policy "Enable update for authenticated users only" on assignments
  for update to authenticated using (true);

create policy "Enable delete for authenticated users only" on assignments
  for delete to authenticated using (true);
```

### 3. Get Credentials
1. Go to **Project Settings** → **API**
2. Copy **Project URL** 
3. Copy **anon** public key 

### 4. Add to Vercel
1. Go to Vercel Dashboard → Your Project → **Settings** → **Environment Variables**
2. Add:
   - `VITE_SUPABASE_URL` = your-project-url
   - `VITE_SUPABASE_ANON_KEY` = your-anon-key
3. Redeploy your project

### 5. Test
- Push code to GitHub
- Vercel will auto-deploy
- Visit your live site - assignments will now sync from Supabase!

- **With Supabase configured**: Data syncs to cloud, visible to all visitors
- **Without Supabase**: Falls back to localStorage (works locally only)
